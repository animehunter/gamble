2 Probabilistic Features

The probabilistic features of this language consist of elementary random procedures (ERPs), a memoizer, and a set of sampler/solvers.

At a high level, every ERP when called produces a value from an associated probability distribution. The precise behavior of an ERP, however, depends on the sampler/solver context it is executed under. In a mh-sampler context, for example, an ERP might reuse a choice from a previous run, subject to random perturbations. In an enumerate context, an ERP will fork its context and potentially the consequences of all values in its distribution.

procedure

(sample dist) → any

  dist : dist?
Returns a sample from dist. The sample procedure cooperates with the enclosing sampler/solver, unlike dist-sample.

procedure

(fail [reason]) → any

  reason : any/c = #f
Used to express condition failure. When used within a sampler or solver, it typically causes the sampler/solver to try again with a different set of choices.

For example, the following function models two coin flips where at least one of them is known to be heads (#t):

(define (flip-two-coins/one-heads)
  (define A (flip))
  (define B (flip))
  (unless (or A B) (fail))
  (list A B))

procedure

(observe-at dist value) → void?

  dist : dist?
  value : any/c
Represents observing the value of a random variable distributed as dist being value. Typically, the effect is to adjust the likelihood of the current sampler/solver execution.

syntax

(observe conditionable-expr value-expr)

Like observe-at, except that instead of a distribution, the observation conditions the result of evaluating conditionable-expr. The conditionable-expr must evaluate to a call to sample (either explicitly or implicitly through one of the random procedures below) in a conditionable context; the call to sample is replaced with a call to observe-at with a suitably adjusted value.

A conditionable context (CC) is (currently) defined as follows:

  CC = [ ]
  | (+ expr ... CC)
  | (cons CC expr)
  | (cons expr CC)
  | (reverse CC)

Support for other invertible built-in functions will be added in the future.

Thus, for example, the following are valid:
> (observe (+ 10 (normal 0 1)) 11.5)

11.5

> (observe (cons (bernoulli) (normal 0 1))
           (cons 0 0.2))

'(0 . 0.2)

> (observe (build-list 3 (lambda (i) (bernoulli)))
           '(1 1 0))

'(1 1 0)

but the following are not:
> (observe (+ (normal 0 1) 10) 11.5)

observe: observation failed

  expected: 11.5

  got: 10.391283542835888

> (observe (vector->list (build-vector 3 (lambda (i) (bernoulli))))
           '(1 1 0))

observe: observation failed

  expected: '(1 1 0)

  got: '(1 0 0)

The call to sample can occur in another function, as long as it occurs in a conditionable context with respect to the function’s body and the function call also occurs within a conditionable context. For example:
> (define (f x) (+ (* 3 x) 12 (normal 0 1)))
> (observe (f 9) 40)

40

Note: because of floating-point imprecision, the result of conditionable-expr may not be exactly equal to value-expr.

procedure

(mem f) → procedure?

  f : procedure?
Returns a memoized version of f.

In general, a memoized function must not be called outside of the dynamic extent of the sampler/solver context in which it was created. See (part "nesting") for more discussion.

Examples:

> (define f (mem (lambda (n) (bernoulli))))
> (f 1)

0

> (f 1) ; calling (f 1) again will get the same value

0

> (for/list ([i 10]) (f i))

'(1 0 0 0 1 0 0 1 1 0)

> (for/list ([i 10]) (f i))

'(1 0 0 0 1 0 0 1 1 0)

2.1 Discrete ERPs

procedure

(flip [p]) → boolean?

  p : (real-in 0 1) = 1/2
Returns #t with probability p, #f with probability (- 1 p).

Equivalent to (= 1 (sample (bernoulli-dist p))).

procedure

(discrete weighted-vals) → any/c

  weighted-vals : (listof (cons/c any/c (>=/c 0)))
Given (discrete (list (cons val weight) ...)), returns a value drawn from the vals with probability proportional to the corresponding weight.

Equivalent to (sample (make-discrete-dist weighted-vals)).

procedure

(discrete* vals weights) → any/c

  vals : (non-empty-listof any/c)
  weights : (non-empty-listof (>/c 0))
Returns a value drawn from vals, with the probability of each element in vals weighted by the corresponding element in weights, if present.

Equivalent to (sample (make-discrete-dist* vals weights)).

2.2 Integer-Valued ERPs

procedure

(bernoulli [p]) → (or/c 1 0)

  p : (real-in 0 1) = 1/2
Returns 1 with probability p, 0 with probability (- 1 p).

Equivalent to (sample (bernoulli-dist p)).

procedure

(binomial count p) → exact-nonnegative-integer?

  count : exact-nonnegative-integer?
  p : (real-in 0 1)
Equivalent to (sample (binomial-dist count p)).

procedure

(categorical weights) → exact-nonnegative-integer?

  weights : (vectorof (>=/c 0))
Equivalent to (sample (categorical-dist weights)).

procedure

(discrete-uniform n) → exact-nonnegative-integer?

  n : exact-positive-integer?
Returns an integer drawn uniformly from [0, n).

Equivalent to (sample (categorical-dist (make-vector n (/ n)))).

procedure

(geometric [p]) → exact-nonnegative-integer?

  p : (real-in 0 1) = 1/2
Equivalent to (sample (geometric-dist p)).

procedure

(poisson mean) → exact-nonnegative-integer?

  mean : (>/c 0)
Equivalent to (sample (poisson-dist mean)).

2.3 Real-Valued ERPs

procedure

(beta a b) → real?

  a : (>/c 0)
  b : (>/c 0)
Equivalent to (sample (beta-dist a b)).

procedure

(cauchy mode [scale]) → real?

  mode : real?
  scale : (>/c 0) = 1
Equivalent to (sample (cauchy-dist mode scale)).

procedure

(exponential mean) → real?

  mean : (>/c 0)
Equivalent to (sample (exponential-dist mean)).

procedure

(gamma [shape scale]) → real?

  shape : (>/c 0) = 1
  scale : (>/c 0) = 1
Equivalent to (sample (gamma-dist shape scale)).

procedure

(logistic [mean scale]) → real?

  mean : real? = 0
  scale : (>/c 0) = 1
Equivalent to (sample (logistic-dist mean scale)).

procedure

(normal [mean stddev]) → real?

  mean : real? = 0
  stddev : (>/c 0) = 1
Equivalent to (sample (normal-dist mean stddev)).

procedure

(pareto scale shape) → real?

  scale : (>/c 0)
  shape : (>/c 0)
Equivalent to (sample (pareto-dist scale shape)).

procedure

(uniform) → real?

(uniform hi) → real?
  hi : real?
(uniform lo hi) → real?
  lo : real?
  hi : real?
Equivalent to (sample (uniform-dist lo hi)).
